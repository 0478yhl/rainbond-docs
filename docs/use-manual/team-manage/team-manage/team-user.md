---
title: 团队成员
description: 介绍团队内成员的管理
---

## 前提条件

* 拥有团队管理员权限

## 添加成员

可将不在此团队内的成员添加到团队内，并分配给其对应的权限。

团队管理 -> 成员 -> 添加成员，选择用户、选择觉得，确认添加即可。

## 修改角色

修改用户在团队下的角色信息。

团队管理 -> 成员 -> 修改角色，选择角色，保存修改。

## 移交团队

将团队管理员的权限移交给其他用户。

团队管理 -> 成员 -> 移交团队，确认保存。

:::danger
移交团队后，你将失去所有权，请谨慎移交团队。
:::

## 删除成员

将团队内的成员删除，这只会将该用户移出此团队，不会删除该用户。

团队管理 -> 成员 -> 删除用户，确认保存。
