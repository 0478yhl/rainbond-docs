云原生应用管理平台

实现微服务架构不用改代码，管理 Kubernetes 不用学容器，帮企业实现应用上云，一站式将任何企业应用持续交付到 Kubernetes 集群、混合云、多云等基础设施。是 Rainstore 云原生应用商店的支撑平台。

产品特性

以应用为中心

应用部署与资源调度解耦合，开发者仅需以最简单的方式描述应用模型，即可完成应用多云部署和高级运维治理，赋能应用交付。

企业中台基座

提供通用应用（组件）构建组装和企业共享，通用服务发布和管理，打通企业IT资源，赋能ToB企业业务中台建设。

Kubernetes 多云管理

Kubernetes 即将成为云原生基础设施，Rainbond 管理多套 Kubernetes 集群提供应用运行环境，屏蔽 Kubernetes 技术复杂性，赋能基础设施管理。

微服务架构

ServiceMesh微服务框架开箱即用，上云即微服务，同时支持其他微服务（SpringCloud, Dubbo）架构协同，赋能企业微服务架构落地。

无侵入 无限制

无侵入支持大多数开发语言持续构建，各种工作负载组件部署，插件化提供微服务治理，支持 Windows 服务，运行于各种云或裸金属，赋能开发者业务开发。

开 源

Rainbond 是一个完整开源的企业级平台。
